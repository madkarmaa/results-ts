# Guide

Welcome to the **results-ts** guide. This library brings Rust's `Result` and `Option` types to TypeScript with full type safety and (_almost_) no runtime overhead - see the [benchmarks](https://github.com/madkarmaa/results-ts/blob/main/BENCHMARKS.md).

- [Getting Started](./getting-started.md) - install the library and understand the basics.
- [Async support](./async.md) - working in the async world.
- [Error handling](./error-handling.md) - panics, misuse errors, and `catchUnwind`.
